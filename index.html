Rocket Game

Description

"Rocket Game" is an exciting game where players control a rocket using their face as the controller. The goal is to avoid enemy missiles colliding with the rocket while accumulating points. The game uses face detection to adjust the rocket's speed and direction, offering a unique and interactive gaming experience.

PC Control with Facial and Voice Recognition

Description

This project consists of a system that allows controlling the mouse and executing commands on a computer using facial and voice recognition. Through head position detection and mouth opening, the user can move the cursor and perform clicks. Additionally, a voice recognition system is implemented to execute commands such as opening programs, adjusting volume, and shutting down the computer.

Technologies Used

  • Python: Programming language used to develop the system.
  • OpenCV: Library for image processing and video capture.
  • MediaPipe: Library for facial mesh detection.
  • PyAutoGUI: Library for controlling the mouse and keyboard.
  • SpeechRecognition: Library for voice recognition.
  • Math: Library for mathematical operations.

Skills Demonstrated

  • Development of interactive applications using Python.
  • Integration of multiple libraries for image processing and voice recognition.
  • Implementation of facial detection algorithms and device control through gestures and voice commands.
  • Problem-solving and real-time code debugging.
  • Design of simple and effective user interfaces using OpenCV.

Sales Management Project

Description

This project is a sales management application that allows users to efficiently control their sales, customers, and inventory. It includes functionalities to register customers, manage inventory, handle payments, and generate sales reports, as well as create invoices in PDF format.

Technologies Used

  • Python: Main programming language used for application development.
  • SQLite: Database management system used to store customer, sales, and inventory information.
  • FPDF: Library used for generating PDF documents, specifically for creating invoices.
  • Flet: Framework used for developing the graphical user interface (GUI).
  • BeautifulSoup: Library used for web scraping and obtaining the dollar price from a web page.

Skills Demonstrated

  • Development of management applications with an interactive graphical interface.
  • Design and management of relational databases using SQLite.
  • Implementation of business logic for sales and payment management.
  • Generation of PDF documents from dynamic data.
  • Data validation and error handling to ensure application robustness.
  • Integration of multiple modules to create a cohesive and functional application.
