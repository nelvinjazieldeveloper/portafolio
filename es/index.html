Juego Cohete

Descripción

"Juego Cohete" es un emocionante juego en el que los jugadores controlan un cohete utilizando su rostro como controlador. El objetivo es evitar que los misiles enemigos colisionen con el cohete mientras se acumulan puntos. El juego utiliza la detección de rostros para ajustar la velocidad y la dirección del cohete, ofreciendo una experiencia de juego única e interactiva.

Tecnologías Utilizadas

  • Python
  • Pygame
  • OpenCV
  • MediaPipe
  • NumPy

Habilidades Demostradas

  • Desarrollo de videojuegos utilizando Pygame.
  • Integración de visión por computadora con OpenCV y MediaPipe.
  • Implementación de detección de colisiones y manejo de eventos en tiempo real.
  • Uso de técnicas de programación orientada a objetos para estructurar el código.
  • Creación de interfaces gráficas interactivas y atractivas.

Control de PC con Reconocimiento Facial y de Voz

Descripción

Este proyecto consiste en un sistema que permite controlar el mouse y ejecutar comandos en una computadora utilizando el reconocimiento facial y de voz. A través de la detección de la posición de la cabeza y la apertura de la boca, el usuario puede mover el cursor y realizar clics. Además, se implementa un sistema de reconocimiento de voz que permite ejecutar comandos como abrir programas, ajustar el volumen y apagar el equipo.

Tecnologías Utilizadas

  • Python: Lenguaje de programación utilizado para desarrollar el sistema.
  • OpenCV: Biblioteca para el procesamiento de imágenes y captura de video.
  • MediaPipe: Biblioteca para la detección de mallas faciales.
  • PyAutoGUI: Biblioteca para controlar el mouse y el teclado.
  • SpeechRecognition: Biblioteca para el reconocimiento de voz.
  • Math: Biblioteca para operaciones matemáticas.

Habilidades Demostradas
  • Desarrollo de aplicaciones interactivas utilizando Python.
  • Integración de múltiples bibliotecas para el procesamiento de imágenes y reconocimiento de voz.
  • Implementación de algoritmos de detección facial y control de dispositivos mediante gestos y comandos de voz.
  • Resolución de problemas y depuración de código en tiempo real.
  • Diseño de interfaces de usuario simples y efectivas utilizando OpenCV.

Proyecto de Gestión de Ventas

Descripción

Este proyecto es una aplicación de gestión de ventas que permite a los usuarios llevar un control eficiente de sus ventas, clientes e inventario. Incluye funcionalidades para registrar clientes, manejar el inventario, gestionar cobros y generar reportes de ventas, así como la creación de facturas en formato PDF.

Tecnologías Utilizadas

  • Python: Lenguaje de programación principal utilizado para el desarrollo de la aplicación.
  • SQLite: Sistema de gestión de bases de datos utilizado para almacenar la información de clientes, ventas e inventario.
  • FPDF: Biblioteca utilizada para la generación de documentos PDF, específicamente para crear facturas.
  • Flet: Framework utilizado para el desarrollo de la interfaz gráfica de usuario (GUI).
  • BeautifulSoup: Biblioteca utilizada para el web scraping y obtener el precio del dólar desde una página web.

Habilidades Demostradas

  • Desarrollo de aplicaciones de gestión con una interfaz gráfica interactiva.
  • Diseño y manejo de bases de datos relacionales utilizando SQLite.
  • Implementación de lógica de negocio para la gestión de ventas y cobros.
  • Generación de documentos PDF a partir de datos dinámicos.
  • Validación de datos y manejo de errores para asegurar la robustez de la aplicación.
  • Integración de múltiples módulos para crear una aplicación cohesiva y funcional.
